Git Repository = https://github.com/tafazul/Pratilipi
Deployed at = https://pratilipiconnect4.herokuapp.com/

Start the game by invoking url 
https://pratilipiconnect4.herokuapp.com/START

Now play the game by sending column in path. The player is switched for alternate valid requests.
https://pratilipiconnect4.herokuapp.com/:column
ex:https://pratilipiconnect4.herokuapp.com/0 ---> to insert in 0th column
   https://pratilipiconnect4.herokuapp.com/1 ---> to insert in 1st column
   https://pratilipiconnect4.herokuapp.com/3 ---> to insert in 3rd column etc
